6.10. Timing – millis()¶

You can find out how many milliseconds (thousands of a second) have elapsed since the run of the program began.

let someTime;

function setup() {
  createCanvas(600,600);
  someTime = 2000;
}

function draw() {
  if (millis() < someTime) {
    background(255);
  }
  else {
    background(0);
  }
}
