p5js Web Editor
=============================

.. topic:: Quick Overview

    As we learn the basics, you will use the p5js web editor. Later on, we will learn to use a more advanced development environment.


If you are just needing to do a quick sketch/experiment using p5js, you can use the `p5js web editor <https://editor.p5js.org/>`_. Although the web editor isn't quite as full featured as setting up a local development environment, it is really simple to jump in and start coding. 

Although you could create a standalone account for the p5js editor, you might find it helpful to use the Login with Google or Login with GitHub buttons.
